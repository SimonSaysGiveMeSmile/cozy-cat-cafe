# 🐾 Cozy Cat Café Run

A sleepy endless runner through the warmest café in town — and you can **swap your runner between cats and memes mid-jump**. Cozy, juicy, shareable.

**Play:** https://cozy-cat-cafe.vercel.app

- 12-character roster (cats 😻😺😸😼🙀😹🐈 + memes 🗿🐸😎🤓💀), each with a distinct feel
- Instant **poof swap** any time — even mid-jump
- Jump / duck obstacles, build the **vibe meter** through COZY → TOASTY → GOLDEN → PURR tiers
- Day → dusk → night drift, screen-juice, persistent hi-score
- Single self-contained `index.html` · vanilla JS + Canvas · zero asset files (emoji + one Google Fonts link)
- Keyboard + touch · `prefers-reduced-motion` aware

Built by [SimonSaysGiveMeSmile](https://github.com/SimonSaysGiveMeSmile).

Controls: **Space/↑/tap** jump · **↓** duck · **C** swap (Shift+C back) · **M** mute · **P** pause
